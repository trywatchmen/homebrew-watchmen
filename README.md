# homebrew-watchmen

Homebrew tap for [WatchmenCLI](https://trywatchmen.cloud) — developer environment intelligence platform.

## Install

```bash
brew tap trywatchmen/watchmen
brew install wm
```

## Upgrade

```bash
brew update
brew upgrade wm
```

## Uninstall

```bash
brew uninstall wm
brew untap trywatchmen/watchmen
```

## Supported platforms

| Platform | Architecture | Status |
|----------|-------------|--------|
| macOS | ARM64 (Apple Silicon) | Native |
| macOS | x86_64 (Intel) | Via Rosetta 2 |
| Linux | x86_64 | Native |
